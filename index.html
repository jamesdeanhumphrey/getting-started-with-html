The World Wide Web

The World Wide Web is a collection of HTML documents. These documents are made up of something called HTML. HTML stands for HyperText Markup Language, or HyperLINKS. Hyperlinks are what connect all pages together.

Major Pieces of The Internet:

You ---> WWW ---> Servers

Your browser makes requests via the Internet to servers using protocol called HTTP. The servers respond with files that your browser displays.

HTML: Hypertext Markup Language

HTML is made up of the following:
-Text content which is what you see.
-Markup is what it looks like.
-Reference to other documents. e.g. Images and Videos
-Links to other pages.

HTML markup is made up of tags:

< opening tag name>contents< /opening> This entire thing is an element.

A few common tags are the "b" tag, "em" tag, "p" tag, "br" tag, and "img" tag.
-The "b" tag is used to make anything that appears inside of the tag itself BOLD.
-The "em" tag turns anything within the tag into ITALICS.
-The "p" tag stands for paragraph, and is a block tag that has content within it like a box.
-The "br" tag is an inline tag that ends a line.
-The "img" tag is used to add pictures within your document, and does not have a cosing tag.

HTML Attributes modify an HTML element, i.g., you can add "a href" within a tag to include a LINK/URL within the document.
A Hyperlink connects web pages to each other.

URL's

URL's are how you refer to documents on the web. It is the unique address for that document on the Internet.

HTTP

In short, HTTP, or HyperText Transfer Protocol, is the set of rules that unites the Internet.

Review Questions:

Q: What is a web page?
A: A web page is a text document written in HTML. Browsers read the docs, interpret and display.

Q: How does coding work?
A: Coding happens when programmers write text in a language that a computer can understand. The computer then follows the instructions.

Q: Are computers stupid?
A: Yes, programmers need to write exactly the way a computer understands. (Also known as the correct syntax)

Q: Can programmers remember everything?
A: Programmers cannot remember everything. There are too many details to keep in your head.

Basic HTML Vocab

Tag: HTML tag is always contained within angled brackets. most have opening and closing tags. Some are called void tags which do not require a closing tag.

Element: Everything between a set of opening and closing tags.

Attribute: Property of an HTML element, i.g., < a href="URL">LINK TEXT< /a>
In short, an attribute modifies an element.
